1.4.1 配置SSH key

SSH key用于解决本地和服务器连接的权限识别和安全问题。
用git bash输入如下命令，查询本机是否已存在SSH密钥

若提示”No such file or directory”说明你是第一次使用SSH key。继续在git bash输入如下命令，生成SSH密钥：

连续3次回车，最终会在本机用户目录下生成一个密钥文件。打开用户目录，找到.ssh\id_rsa.pub文件，使用记事本打开并复制里面的内容。
打开你的github首页，点击右上角用户头像，点击下拉菜单中的”Settings”

在加载的页面中选择左侧”SSH and GPG Keys”，点击”New SSH key”，将上面复制的SSH key粘贴至”Key”文本框中，”Title”随意填，点击保存。
